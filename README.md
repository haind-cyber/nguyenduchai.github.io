# Landing Page Nguyễn Đức Hải

Landing page chuyên nghiệp với trọng tâm MC tiệc cưới (chiếm 50% trang chủ).

## Đặc điểm nổi bật

1. **Bố cục 50-50**:
   - 50% cho mảng MC (Banner + Video)
   - 50% cho các mảng khác (CNTT, Sinh trắc, Đầu tư)

2. **Bài viết chào mừng**:
   - Bài viết "Chào bạn, tôi là Nguyễn Đức Hải!" với giọng văn thân thiện, chuyên nghiệp

3. **Thiết kế hiện đại**:
   - Màu sắc chuyên nghiệp (hồng chủ đạo cho MC)
   - Responsive trên mọi thiết bị
   - Animation mượt mà

## Hướng dẫn triển khai GitHub Pages

1. Tạo repository trên GitHub với tên: `nguyenduchai.github.io`
2. Upload 4 file này vào repository
3. Vào Settings → Pages → Chọn branch `main` → Save
4. Truy cập: `https://nguyenduchai.github.io`

## Tùy chỉnh cần thiết

1. **Ảnh MC**:
   - Thay `banner-placeholder` bằng ảnh MC thật
   - Kích thước khuyến nghị: 2000x600px

2. **Video MC**:
   - Upload video lên YouTube/Vimeo
   - Thay `video-placeholder` bằng embed code

3. **Thông tin liên hệ**:
   - Cập nhật email, số điện thoại thật
   - Cập nhật link mạng xã hội

4. **Màu sắc**:
   - Tùy chỉnh biến màu trong `style.css`

## Cấu trúc file
- `index.html` - Trang chính
- `style.css` - Styling
- `script.js` - JavaScript
- `README.md` - Hướng dẫn

## Tích hợp form liên hệ thực tế

Để form liên hệ hoạt động thực tế, bạn cần:
1. Tạo tài khoản Formspree/Netlify Forms
2. Thay action của form trong `index.html`
3. Hoặc sử dụng Google Forms
